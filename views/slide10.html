Loading The Data

I saved the link as a CSV file called data.csv. D3 comes equipped with a CSV parsing function: d3.csv("data.csv", callback). We'll put most of our code inside of the callback function because we can't draw anything until the data loads.

d3.csv() turns our CSV into an array of objects. That way each row of the CSV is one point of data with the approriate object properties for each column.

[
 {activity: "active", batch: "Batch_01", class: "Alphaproteobacteria", clone: ... },
 {activity: "work stopped", batch: "Batch_01", class: "Alphaproteobacteria", clone: ... },
 {activity: "active", batch: "Batch_01", class: "Alphaproteobacteria", clone: ... },
