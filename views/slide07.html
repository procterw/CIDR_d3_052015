D3 Basics: CSS

If you're not planning on downloading your D3 plot, most of the styling should be done with CSS. The following CSS made the elements on slide 5 blue.

circle, rect, ellipse, polygon {
  fill: #3498db;
}

line, polyline {
  fill: none;
  stroke: #3498db;
  stroke-width: 2;
}

You can also add classes to SVG elements.

circle.orange { fill: orange; }
circle.blue { fill: blue; }

<circle cx="40" cy="50" r="25" class="orange"/>
<circle cx="100" cy="50" r="25" class="blue"/>
