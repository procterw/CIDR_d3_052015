What Is D3 Not

A charting library. You can't pass data to a built in function and get back a nice graphic.

A graphics library. D3 is an API for maniupulating the browser's DOM.

What Is D3 Not So Good For?

Exploratory graphics. Have an idea what you want the visualization to look like before you start.

Dealing with huge datasets. Too many elements (~several thousand) will slow the browser down to a crawl.
