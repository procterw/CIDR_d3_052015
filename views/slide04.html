D3 Basics: SVG

D3 usually usues SVG, an XML based image vector format. All modern web browsers support SVG.

Since SVG elements are vectors, they scale with no issue. Great for scientific graphics and text.

<svg>
  <rect width="100" height="100" transform="rotate(45) translate(50,-50)"/>
  <circle r="25" cx="70" cy="70" />
</svg>
