What Is D3

D3.js is a JavaScript library for manipulating documents based on data. D3 helps you bring data to life using HTML, SVG, and CSS.

In other words, each point of data is linked to a DOM element.

What Is D3 Good For?

Creating powerful interactive data visualizations.

Creating unique visualizaions that can't be easily made in other software.

Procesing and manipulating data in Javascript.

Visualizations are typically in SVG (scalable vector graphics), which can be downloaded and treated like any other SVG or PDF file.
