Enter, Update, and Exit

The plotting code in D3 is typically wrapped in a function, which we'll call plot.

plot() {
  
  filter and nest dataset

  update y and x1 scales

  update y axis

  update bar groups
  enter bar groups
  exit bar groups

  update rectangles
  enter rectangles
  exit rectangles

}

You'll notice the two sections of update, enter, and exit. In dynamic D3 visualizatons this is called the general update pattern.

enter We've already used enter several times. It adds DOM elements until there are enough for each data point.

update Reposition elements that already have data bound to them. This won't be called the first time.

exit If some of the data is removed, exit will remove elements until there is one per data point.
