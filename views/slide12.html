Conceptually Structuring The Data

First let's think about what the data needs to look like to make our barchart. We need to plot by status, sort by maxCode, filter by several different attributes and group by several different attributes.

We want to use different filters and grouping, but the data will always need to be sorted by maxcode, so let's do that now.

d3.csv("data.csv", function(dataset) {

  dataset = _.sortBy(dataset, function(d) { return parseInt(d.maxCode); });

});

The _.sortBy() is from lodash, a super useful JS library which has lots of useful functions for manipulating data. It'll come up frequently in these examples.
